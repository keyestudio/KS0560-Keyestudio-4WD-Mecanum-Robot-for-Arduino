### Wiring Diagram

**The wiring of the ultrasonic module**

| ultrasonic | Sensor expansion board |
| ---------- | ---------------------- |
| Vcc        | V                      |
| Trig       | D12                    |
| Echo       | D13                    |
| Gnd        | G                      |

![](media/wps32.png)

![](media/wps33.png)

**The wiring of the servo**

| Servo  | Sensor expansion board |
| ------ | ---------------------- |
| Brown  | G                      |
| Red    | 5V                     |
| Orange | D9                     |

![](media/wps34.jpg)

![](media/wps35.png)

| S4           | D10             |

![](media/wps37.png)

| Driver Board | Expansion Board |
| ------------ | --------------- |
| SCL          | D2              |
| SDA          | D3              |
| 5V           | D5              |
| GND          | D4              |

| S3           | A0              |
| 5V           | 5V              |
| GND          | G               |
